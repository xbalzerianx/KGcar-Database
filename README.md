# 🚗 KGCAR Product Database

A premium web-based product inventory management system for KGCAR.

## Features
- 📊 **Dashboard** — Live inventory value, sales totals, low stock alerts, category breakdown
- 📦 **Products** — Full product catalog with search, filter, and stock status
- 💰 **Record Sale** — Log a sale and auto-deduct stock in one click
- 📋 **Sales Log** — Full transaction history with search and filters
- ⚠️ **Stock Alerts** — Instant view of low stock and out of stock items
- 📱 **Mobile Optimized** — Bottom tab navigation, card layouts for small screens

## Tech Stack
- Vanilla HTML/CSS/JS (single file, zero dependencies)
- Base44 SDK for database (entities: KGProduct, KGSale)
- Dark navy + gold premium design theme

## Categories
Tires · Engine Oil · Brake Pads · Brake Oil · Filters · Batteries · Wipers · Lights · Belts & Hoses · Accessories

## Setup
1. Open `index.html` in a browser or deploy to any static host
2. Powered by Base44 backend — data persists automatically
